À propos

Passionné par
le web & le code

Je suis Onelson Cadeau, développeur Front-End junior formé chez OpenClassrooms. Je crée des interfaces web modernes, responsives et accessibles, avec une attention particulière à l'expérience utilisateur.

Curieux et rigoureux, je m'appuie sur HTML5, CSS3, JavaScript ES6+ et React pour donner vie à des projets ambitieux. Chaque ligne de code est pour moi une opportunité de construire quelque chose de meilleur.
